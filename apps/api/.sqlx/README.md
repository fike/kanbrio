# SQLx Query Journal

This directory contains SQLx query metadata generated by `cargo sqlx prepare`.

## Usage

### 1. Generate the journal

After changes to SQL queries or database migrations:

```bash
make sqlx-prepare
```

### 2. Commit the journal

The `.sqlx/` files **must be committed** so CI can verify queries without a live database.

```bash
git add apps/api/.sqlx/
git commit -m "chore(sqlx): update query journal"
```

### 3. Enable offline mode in CI

```yaml
env:
  SQLX_OFFLINE: "true"
```

## Story #19

All queries use `sqlx::query!`, `sqlx::query_as!`, or `sqlx::query_scalar!`
for compile-time verification.
